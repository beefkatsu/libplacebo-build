# libplacebo-build

build scripts for [libplacebo](https://github.com/haasn/libplacebo)

> This is a component of the [MPVKit](https://github.com/mpvkit/MPVKit) project.

## Installation

### Swift Package Manager

```
https://github.com/mpvkit/libplacebo-build.git
```

## How to build

```bash
make build
# or build specified platforms
make build platform=ios,macos
# or see help
make help
```
